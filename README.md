## English

Principles and best practices for AI assistants to follow proactively during planning, code generation, and problem-solving. Language-agnostic, applicable to any tech stack.

### Principles

1. **Package-First Approach** - Prefer mature open-source libraries over custom implementations
2. **UX for Long-Running Operations** - Design failure-prone or lengthy operations with progress tracking, fault tolerance, and resumability
3. **Leverage Framework & Library CLI Tools** - Research available CLI tools and make case-by-case decisions

## 中文

为 AI 助手提供通用的工作流原则，帮助 AI 在计划、代码生成和问题解决各阶段做出更好的决策。适用于任何编程语言和技术栈。

### 包含原则

1. **包优先原则** - 优先使用成熟的开源包，而不是自己实现
2. **长流程操作的用户体验设计** - 为耗时操作提供进度反馈、断点恢复等机制
3. **利用框架和库的 CLI 工具** - 了解可用的 CLI 工具，根据情况选择使用
